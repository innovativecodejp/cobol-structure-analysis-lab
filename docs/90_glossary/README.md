# Glossary (Terminology Definitions)

This directory contains terminology definitions for the COBOL Structure Analysis Lab.

## File Structure

- `Glossary.md`: Main glossary index and general terms.
- `TermsAstIrCfgDfg.md`: Terms related to AST, IR, CFG, and DFG (Syntax/Structure layers).
- `TermsGuaranteeSpace.md`: Terms related to Guarantee Space and Migration Geometry (Guarantee/Geometry layers).

## Term Format

```

**Layer**: Syntax / Structure / Guarantee / Geometry / Decision

**Definition**:
A clear, concise definition of the term.

**Formal Description**:
Mathematical or formal definition (if applicable).
$$
Formula
$$

```
