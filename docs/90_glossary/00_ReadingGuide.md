# Glossary 0. Reading Guide

**Status**: Defined
**Layer**: Meta

## Definition
本用語集（Glossary）の構成と読み方を定義するガイド。
本研究において「用語」は、単なるラベルではなく、理論モデルの構成要素（Component of Theory）として厳密に定義される。

## Structure
用語集は以下のカテゴリに分類される。

1.  **Core Concepts**: 研究全体を貫く最重要概念。
2.  **Structural Analysis Layer**: 構文・意味解析に関連する概念（AST, IR, CFG, DFG）。
3.  **Guarantee Theory**: 保証（Guarantee）の定義と構成要素。
4.  **Guarantee Space Theory**: 保証空間の数理モデル。
5.  **Geometry of Migration**: 移行の幾何学的解釈。
6.  **Structural Modeling**: 構造解析の基礎概念。
7.  **Migration Decision Model**: 意思決定とリスクモデル。
8.  **Control Structure Model**: 制御フローのモデル化。
9.  **Data Structure Model**: データ構造のモデル化。
10. **Lattice / Order Theory**: 順序理論・束論の基礎概念。
11. **Derived Metrics**: 構造から導出される計量指標。

## Notation
各用語は以下のフォーマットで記述される。

*   **Term**: 用語名
*   **Layer**: 抽象レイヤ（構文層 / 構造層 / 判断層 / 理論層）
*   **Definition**: 定義文
*   **Formal Description**: 数学的または形式的な定義（該当する場合）
*   **Related Concepts**: 関連する他の用語
*   **Example**: 具体例

## Status
各用語定義の状態。
*   `Defined`: 定義済み
*   `Draft`: 草案
*   `Placeholder`: 定義待ち（構造のみ存在）
