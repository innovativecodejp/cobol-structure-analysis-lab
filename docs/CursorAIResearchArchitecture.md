# Cursor AI研究体制（AI千日回峰行版）

## 目的

AI千日回峰行の研究（Guarantee Space / Migration Geometry /
移行設計理論）を\
Cursor + 複数AIで効率的に進めるための研究体制を定義する。

AIを単なるツールとして使うのではなく\
**研究チームとして役割分担させる**ことを基本思想とする。

------------------------------------------------------------------------

# AI研究チーム構成

研究者（あなた）\
↓\
AI研究チーム

Gemini 3 Pro → 研究設計\
Sonnet 4.1 → 研究レビュー\
ChatGPT Codes → 実装\
Composer → 軽作業

------------------------------------------------------------------------

# 1 研究設計AI

## Gemini 3 Pro

### 役割

Chief Research Architect（研究設計責任者）

### 担当領域

-   研究テーマ整理
-   概念設計
-   Guarantee Space理論整理
-   Migration Geometry構造設計
-   研究ロードマップ作成
-   Mermaid図生成
-   研究ログ分析
-   論理構造整理

### 特徴

Geminiは - 長文理解が強い - 概念整理が得意 - 数理構造整理が得意

AI千日回峰行の研究において **最も重要なAI**。

### 主な使用場面

-   研究ログ分析
-   理論整理
-   研究方向の決定
-   Mermaid図生成
-   研究ロードマップ作成

------------------------------------------------------------------------

# 2 研究レビューAI

## Sonnet 4.1

### 役割

Research Auditor（研究監査）

### 担当領域

-   研究の矛盾チェック
-   論理の破綻チェック
-   定義の曖昧さ検出
-   Mermaid図の検証
-   ドキュメント整形
-   論理レビュー

### 役割の意味

Gemini：新しいアイデアを作る\
Sonnet：論理の穴を見つける

この役割分担が研究の品質を高める。
