# 1 Guarantee Measure

重み関数

w : P → ℝ⁺

を定義する。

保証強度

Strength(S) = Σ w(p)

を定義する。

---

## 要求

Strength を

**Measure**

として整理せよ。

μ : P(P) → ℝ

μ(S) = Σ w(p)

以下を示せ。

1 μ(∅) = 0  
2 S ∩ T = ∅ → μ(S ∪ T) = μ(S) + μ(T)

# 2 Guarantee Metric

距離

d(G1,G2) = |G1 △ G2|

を定義する。

これは

**Hamming metric**

である。

# 3 Weighted Metric

d_w(G1,G2) = Σ w(p)

**Weighted Hamming metric**
