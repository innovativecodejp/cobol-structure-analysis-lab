# Phase1 Prompt01

## Hypercube / Dependent Guarantee Space

### Goal

Define the structural model of Guarantee Space and its
dependency-restricted subspace.

### Task

1.  Define Guarantee Space

G = P(P)

Interpret it as:

-   Power set
-   Boolean lattice
-   N-dimensional hypercube

2.  Introduce dependency relation

D ⊆ P × P

Define partial order:

p_j ≤\_D p_i (p_i depends on p_j)

3.  Define Dependent Guarantee Space

G_dep = Idl(P, ≤\_D)

Where Idl(P, ≤\_D) are ideals (lower sets) of the poset.

4.  Explain that:

-   G_dep ⊆ P(P)
-   G_dep forms a finite distributive lattice.
