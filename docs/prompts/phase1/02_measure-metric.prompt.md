# Phase1 Prompt02

## Measure / Metric on Guarantee Space

### Goal

Introduce quantitative structure into Guarantee Space.

### Task

1.  Define weight function

w : P → ℝ≥0

2.  Define measure on guarantee state

μ(S) = Σ w(p) for p ∈ S

3.  Define symmetric difference

S △ T = (S  T) ∪ (T  S)

4.  Define metric

d(S,T) = μ(S △ T)

5.  Verify metric properties

-   Non‑negativity
-   Identity
-   Symmetry
-   Triangle inequality
