# Phase1.5 Prompt02

## Measure / Metric on Guarantee Space

### Goal

Introduce quantitative geometry to Guarantee Space.

### Tasks

1.  Define weight function

w : P → ℝ≥0

2.  Define measure

μ(S) = Σ w(p) for p ∈ S

3.  Define symmetric difference

S △ T = (S  T) ∪ (T  S)

4.  Define metric

d(S,T) = μ(S △ T)

5.  Verify metric axioms

-   Non-negativity
-   Identity of indiscernibles
-   Symmetry
-   Triangle inequality

Explain that this corresponds to a weighted Hamming metric on the
hypercube.
