# Prompt02: Migration Complexity

## Goal

Define the mathematical difficulty of migration.

## Task

Using the Guarantee Transition Graph:

### Migration Distance

d(S,T) = \|T  S\|

### Guarantee Weight

w(p) ≥ 0

### Path Cost

Cost(Path) = Σ w(p)

### Minimum Migration Cost

Find the minimum cost path from ⊥ to ⊤.

### Migration Complexity

C = minimum migration cost
